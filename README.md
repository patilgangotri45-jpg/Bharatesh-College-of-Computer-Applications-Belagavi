# Bhartesh College of Computer Applications Belagavi Event Registration and Management System

## 👩‍💻 Developed By

Gangotri Patil

## 📖 Description

This project is a web-based Event Registration and Management System developed for Bhartesh College of Computer Applications, Belagavi.

The system allows administrators to create events, manage registrations, generate QR codes, monitor active registrations, maintain deleted registration history, and manage uploaded event photos.

Students can register for events, download their QR codes, and upload event photos after registration.

## ✨ Features

- Admin Dashboard
- Event Creation
- Student Registration
- QR Code Generation
- QR Code Download
- Active Events
- Active Registrations
- Registration Deadline
- Registration Delete Deadline
- Deleted Registration History
- Event Photo Upload
- Admin Photo Gallery
- Local Storage Support

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage
- QRCode.js
